# show up Flutter prototype

웹 프로토타입을 바탕으로 만든 Flutter 화면 초안입니다.

포함 화면:
- 인트로, 로그인, 회원가입, 비밀번호 찾기
- Home 피드
- Camera 참가 흐름
- Vote 투표
- Bet 예측
- Rank 랭킹

실행:

```powershell
cd "C:\Users\김영진\Documents\어플리케이션\showup_flutter"
flutter pub get
flutter run
```

현재는 서버, 실제 카메라, 실제 광고, 실제 영상 업로드가 붙기 전의 UI/흐름 프로토타입입니다.
